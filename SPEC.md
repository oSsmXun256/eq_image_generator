# 震源・震度画像 API 仕様書


この仕様書のバージョン： Ver1.0.0	2026-05-01

---

# 概要
地震情報を与えると震源マップ画像を生成・返却するAPIです。

エンドポイント: `GET https://your-domain.example/eqimg/`

レスポンス形式: JPEG 画像 (`Content-Type: image/jpeg`)

---

# URL パラメーター

## パラメーター一覧

| パラメーター | 型     | 必須 | 説明                                                                 | 例            |
|-------------|--------|------|----------------------------------------------------------------------|---------------|
| `type`      | string | ✅   | 速報種別。`eew`（緊急地震速報）または `eqlist`（地震情報）           | `eew`         |
| `lat`       | number | ✅   | 震源の緯度（10進法）                                                 | `35.6895`     |
| `lng`       | number | ✅   | 震源の経度（10進法）                                                 | `139.6917`    |
| `int`       | string | ✅   | 最大震度ラベル（下記の震度ラベル一覧を参照）                         | `5弱`         |
| `mag`       | number | ✅   | マグニチュード                                                       | `6.5`         |
| `dep`       | string | ✅   | 震源の深さ（km）                                                     | `10`          |
| `loc`       | string | ✅   | 震源地名（URL エンコード必須）                                       | `茨城県北部`  |
| `serial`    | number | —    | 速報回数（EEW のみ。省略時は表示なし）                               | `3`           |
| `final`     | 0 \| 1 | —    | 最終報フラグ（`1` = 最終報。省略時は `0` 扱い）                      | `1`           |

---

## 震度ラベル一覧（`int` パラメーター）
## 震度ラベル一覧

| ラベル | 意味     |
|--------|----------|
| `1`    | 震度1    |
| `2`    | 震度2    |
| `3`    | 震度3    |
| `4`    | 震度4    |
| `5弱`  | 震度5弱  |
| `5強`  | 震度5強  |
| `6弱`  | 震度6弱  |
| `6強`  | 震度6強  |
| `7`    | 震度7    |

## URL 構築例
EEW（緊急地震速報）の場合
```
GET https://your-domain.example/eqimg/?type=eew&lat=35.6895&lng=139.6917&int=5%E5%BC%B1&mag=6.5&dep=10&loc=%E8%8C%A8%E5%9F%8E%E7%9C%8C%E5%8C%97%E9%83%A8&serial=3&final=0
```
EQList（気象庁 地震情報）の場合
```
GET https://your-domain.example/eqimg/?type=eqlist&lat=35.6895&lng=139.6917&int=4&mag=5.2&dep=30&loc=%E5%9F%BC%E7%8E%89%E7%9C%8C%E5%8D%97%E9%83%A8&final=1
```

# 画像仕様

## 画像仕様

| 項目                   | 内容                                                                 |
|------------------------|----------------------------------------------------------------------|
| フォーマット           | JPEG                                                                 |
| 推奨サイズ             | 1600 × 900 px（16:9）                                                |
| 背景                   | 日本地図（全国または周辺地域にズームイン）                           |
| 震源マーク             | バツ印（震度に応じた色）                                             |
| 震度色帯               | `INTENSITY_COLOR` と同じカラーコードを使用                           |
| テキストオーバーレイ   | 震源地名・震度・M・深さ・発生時刻（任意）                            |
| 最終報                 | `final=1` のとき「最終報」バッジを右上に表示                         |
